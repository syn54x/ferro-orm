    Core Models¶

    Model ¶

    Bases: BaseModel

    Provide the base class for all Ferro models

    Inheriting from this class registers schema metadata with the Rust core and exposes high-performance CRUD and query entrypoints.

    Examples:

    >>> class User(Model):
...     id: int | None = None
...     name: str
